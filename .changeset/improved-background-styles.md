Improve background style handling in Tasty. Add `image` style for background images. The `fill` handler now supports all background CSS properties (`backgroundPosition`, `backgroundSize`, `backgroundRepeat`, `backgroundAttachment`, `backgroundOrigin`, `backgroundClip`). Add `background` and `image` transition semantic names. Deprecate `backgroundColor`, `backgroundImage`, and `background` styles in favor of `fill`, `image`, and separate background properties.
