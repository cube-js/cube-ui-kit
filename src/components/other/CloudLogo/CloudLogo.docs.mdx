# CloudLogo

A Cube Cloud logo button. Extends `Button` with `type="clear"` and brand styling. Used for navigation branding in app headers.

## When to Use

- App header branding
- Navigation to home/dashboard

## Properties

Extends [Button](/docs/actions-button--docs) props (`onPress`, `to`, `href`, etc.).

### Style Defaults

- `type` — `clear`
- `padding` — `0`
- `gap` — `0`
- `fill` — `#clear`
- `border` — `0`
- `radius` — `0`

### Base Properties

Supports [Base properties](/docs/base-properties--docs).

## Examples

```jsx
<CloudLogo onPress={() => navigate('/')} />

<CloudLogo to="/" />
```
