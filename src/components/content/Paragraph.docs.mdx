# Paragraph

A block-level text component for body content. Renders as `<p>` and extends `Text` with paragraph-specific defaults.

## When to Use

- Body text and descriptions
- Multi-line content that should flow as paragraphs

## Properties

Extends [Text](/docs/generic-text--docs) properties including `preset`, `color`, `weight`, `transform`, `ellipsis`, `nowrap`.

### Style Defaults

- `display` — `block`
- `preset` — `p3`
- `color` — `#dark-02`

### Base Properties

Supports [Base properties](/docs/getting-started-base-properties--docs).

## Examples

```jsx
<Paragraph>This is a paragraph of body text.</Paragraph>

<Paragraph preset="p2" color="#minor">
  Secondary paragraph with custom styling.
</Paragraph>
```
