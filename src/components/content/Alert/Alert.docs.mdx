# Alert

An alert message with theme variants. Used for announcements, warnings, and informational messages.

## When to Use

- Form validation messages
- Success/error/warning notifications
- Informational banners

## Properties

- **`theme`** `'note' | 'success' | 'danger' | 'warning' | 'disabled'` (default: `'note'`) — Visual theme
- **`type`** — Deprecated, use `theme` instead
- **`shape`** `'card' | 'sharp'` (default: `'card'`) — `card` shows border radius and border; `sharp` removes both

### Style Defaults

- `display` — `block`
- `flow` — `column`
- `padding` — `1.5x`
- `preset` — `t3`
- `radius` — `1cr` (switches to `0` when `shape="sharp"`)
- `color` — `#dark-02`

### Base Properties

Supports [Base properties](/docs/getting-started-base-properties--docs).

### Style Properties

These properties allow direct style application without using the `styles` prop:

- **Base:** `display`, `font`, `preset`, `hide`, `whiteSpace`, `opacity`, `transition`
- **Position:** `gridArea`, `order`, `gridColumn`, `gridRow`, `placeSelf`, `alignSelf`, `justifySelf`, `zIndex`, `margin`, `inset`, `position`
- **Dimension:** `width`, `height`, `flexBasis`, `flexGrow`, `flexShrink`, `flex`
- **Block:** `border`, `radius`, `shadow`, `outline`, `padding`, `paddingInline`, `paddingBlock`, `overflow`, `scrollbar`, `textAlign`
- **Color:** `color`, `fill`, `fade`, `image`
- **Text:** `textTransform`, `fontWeight`, `fontStyle`
- **Flow:** `flow`, `placeItems`, `placeContent`, `alignItems`, `alignContent`, `justifyItems`, `justifyContent`, `align`, `justify`, `gap`, `columnGap`, `rowGap`, `gridColumns`, `gridRows`, `gridTemplate`, `gridAreas`

## Examples

```jsx
<Alert theme="note">Informational message</Alert>

<Alert theme="success">Operation completed successfully</Alert>

<Alert theme="danger">Something went wrong</Alert>

<Alert theme="warning" shape="sharp">Warning with sharp corners</Alert>
```
