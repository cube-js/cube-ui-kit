# Content

A scrollable content section for page layouts. Renders as `<section>` and is designed to work with `Layout`.

## When to Use

- Main content area in app layouts
- Scrollable body sections
- Content that flows with `Layout` header/footer

## Properties

No component-specific props. Use style props directly.

### Style Defaults

- `display` — `block`
- `flow` — `column`
- `gap` — `2x`
- `preset` — `p3`
- `color` — `#dark-02`
- `overflow` — `auto`
- `scrollbar` — `styled`
- `gridArea` — `content`

### Base Properties

Supports [Base properties](/docs/getting-started-base-properties--docs).

### Style Properties

These properties allow direct style application without using the `styles` prop:

- **Base:** `display`, `font`, `preset`, `hide`, `whiteSpace`, `opacity`, `transition`
- **Position:** `gridArea`, `order`, `gridColumn`, `gridRow`, `placeSelf`, `alignSelf`, `justifySelf`, `zIndex`, `margin`, `inset`, `position`
- **Dimension:** `width`, `height`, `flexBasis`, `flexGrow`, `flexShrink`, `flex`
- **Block:** `border`, `radius`, `shadow`, `outline`, `padding`, `paddingInline`, `paddingBlock`, `overflow`, `scrollbar`, `textAlign`
- **Color:** `color`, `fill`, `fade`, `image`
- **Text:** `textTransform`, `fontWeight`, `fontStyle`
- **Flow:** `flow`, `placeItems`, `placeContent`, `alignItems`, `alignContent`, `justifyItems`, `justifyContent`, `align`, `justify`, `gap`, `columnGap`, `rowGap`, `gridColumns`, `gridRows`, `gridTemplate`, `gridAreas`

## Examples

```jsx
<Layout>
  <Layout.Header>
    <Title level={3}>Page Title</Title>
  </Layout.Header>
  <Layout.Content>
    <Paragraph>Main page content goes here.</Paragraph>
  </Layout.Content>
  <Layout.Footer>
    <Text>Footer</Text>
  </Layout.Footer>
</Layout>
```
