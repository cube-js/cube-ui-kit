# CopyPasteBlock

A block that displays a value with copy-to-clipboard functionality. Supports paste via click/focus.

## When to Use

- API keys or tokens display
- One-time codes or passwords
- Values users need to copy or paste

## Properties

- **`title`** `string` ‚Äî Label for the block
- **`value`** `string` (default: `''`) ‚Äî The value to display and copy
- **`placeholder`** `ReactNode` ‚Äî Placeholder when value is empty
- **`size`** `'small' | 'medium' | 'large'` (default: `'medium'`) ‚Äî Block size
- **`onPaste`** `(text: string) => void | Promise<void | string>` ‚Äî Callback when text is pasted into the block
- **`onCopy`** `() => void` ‚Äî Callback when the value is copied from the block

### Style Defaults

- `fill` ‚Äî `#dark-bg`
- `radius` ‚Äî `1r`
- `cursor` ‚Äî pointer
- `preset` ‚Äî `t3` (switches to `t2` when `size="large"`)
- `height` ‚Äî `5x` (`4x` for small, `6x` for large)

### Base Properties

Supports [Base properties](/docs/getting-started-base-properties--docs).

## Examples

```jsx
<CopyPasteBlock title="API Key" value={apiKey} />

<CopyPasteBlock title="Token" value={token} size="small" />
```
