## Flow layout

Flow Component is used to create gaps between blocks inside block layout.

```jsx
<Card gap>
  <Card>
    Card 1
  </Card>
  <Card>
    Card 2
  </Card>
  <Card>
    Card 3
  </Card>
</Flow>
```

<Card gap>
  <Card>
    Card 1
  </Card>
  <Card>
    Card 2
  </Card>
  <Card>
    Card 3
  </Card>
</Card>

## Flex layout

Use Space Component to create a row or a column of elements.

```jsx
<Space flow="row">
  <Block>horizontal space</Block>
  <Button>One</Button>
  <Button>Two</Button>
</Space>
<Space flow="column">
  <Block>vertical space</Block>
  <Button>One</Button>
  <Button>Two</Button>
</Space>
```

<Card gap>
  <Space flow="row">
    <Block>horizontal space</Block>
    <Button>One</Button>
    <Button>Two</Button>
  </Space>
  <Space flow="column">
    <Block>vertical space</Block>
    <Button>One</Button>
    <Button>Two</Button>
  </Space>
</Card>

Use Flex Component for more flexible layouts.

```jsx
<Flex flow="row" placeContent="space-between" placeItems="center">
  <Button>One</Button>
  <Button>Two</Button>
  <Button>Three</Button>
  <Button>Four</Button>
</Flex>
```

<Card>
  <Flex flow="row" gap="4x" placeContent="space-between" placeItems="center">
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
    <Button>Four</Button>
  </Flex>
</Card>

## Grid layout

Use Grid Component for grid layouts.

### Simple layout with side bar.

```jsx
<Grid gridColumns="200px 1fr" gridRows="60px 1fr" height="500px">
  <Card gridColumn="1 / -1">Header</Card>
  <Card>Sidebar</Card>
  <Card>Content</Card>
</Grid>
```

  </Grid>
</Card>
